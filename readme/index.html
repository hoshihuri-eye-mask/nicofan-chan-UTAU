    UTAU利用規約(2025/03/31)

    星降りアイマスクの管理するUTAU音源の利用規約となります。

    ご利用の際には以下の規約に同意されたと見なされます。

    音源利用規約

    禁止事項

    • 当音源の二次配布や無断転載、無断アップロード
    • 当音源の自作発言
    • 宗教非難、勧誘、布教または政治活動、批判、主張を目的とした使用
    • 当音源の別音源への流用
    • 作品投稿先の規定、法律に反した使用
    • 併用他音源又は素材の規定に干渉する使用
    • 当音源の悪意ある改変、加工
    • 当音源を使用してのAI学習企業、法人による使用、商用利用

    許可事項

    • ニコニコ動画、YouTubeなど動画共有サービスが提供する機能での収益化、ストリーミング配信
    • 楽曲のカバー、オリジナル楽曲での音源使用
    • 二次創作(*1)
    • 他音源との併用(他音源の規約に違反しないよう確認をお願いします)
    • 同人利用など非営利目的のCD、グッズ頒布(報告は任意)
    • ノイズ除去や周波数表の修正等の調整

    (*1)性的/暴力/グロテスクな表現に関しては投稿先の規約にしたがい、かつワンクッションや検索避けをお願いいたします。また、それぞれのサイトやプラットフォームのルールに従って下さい。

    立ち絵利用規約

    禁止事項

    • 二次配布や無断転載、無断アップロード
    • 自作発言
    • イラストのトレース
    • 宗教非難、勧誘、布教または政治活動、批判、主張を目的とした使用
    • 当キャラクター、他者の製作物、個人、団体への中傷や誹謗を目的とした使用
    • 他のキャラクターとしての使用、改変（TRPG動画等でのロールプレイは可）
    • 作品投稿先の規定、法律に反した使用
    • 悪意ある改変、加工
    • 当立ち絵を使用したグッズの制作
    • 当立ち絵を使用したSNS等でのアイコン、ヘッダー使用
    • 当サイトで公開、配布している立ち絵を使用してのAI学習
    • 企業、法人による使用、商用利用

    許可事項

    1. ニコニコ動画、YouTubeなど動画共有サービスが提供する機能での収益化、ストリーミング配信
    2. 当音源を使用したカバー楽曲、オリジナル楽曲での公式立ち絵の使用

    免責事項

    • 当音源、キャラクターを使用して生じたトラブルに関しては一切の責任を負いません。
    • 当音源、キャラクターを使用する際はこちらの利用規約に同意されたものと見なします。
    • 利用規約違反が確認された際は、音源、立ち絵の使用を禁止させて頂く場合があります。

    また、この規約文は追加,変更される場合があります。

    必ず最新の利用規約に従ってください。
